# 事業継続力強化計画（BCP）策定アプリ 操作マニュアル (商工会職員編)

本マニュアルは、商工会職員様が本アプリを使用して、経営者から提出された事業継続力強化計画データを確認・添削し、申請支援を行うための手順書です。

---

## 🛑 はじめに：支援の流れ

1. **データの受領**: 経営者から計画データ（JSONファイル）を受け取ります。
2. **データの読み込み**: 本アプリにデータを読み込ませ、現状の完成度や不足項目を確認します。
3. **確認・指導**: 不足している項目や、具体性が足りない部分についてアドバイスを行います。
4. **申請**: 完成した計画書をもとに、電子申請のサポートを行います。

---

## 1. データの読み込み（インポート）

経営者から受け取ったJSONファイル（`jigyokei_data_xxxx.json`）をご自身のアプリ環境に読み込みます。

1. アプリにログインします。
2. 画面左側の**「サイドバー」**を下へスクロールします。
3. **「Data Management」**（データ管理）という項目を探します。
4. 「📂 **JSONファイルをドラッグ＆ドロップ**」というエリアに、受け取ったJSONファイルをドラッグして入れます。
5. 画面右上に「✅ 事業計画データを読み込みました」というメッセージが出れば成功です。
   * これで、経営者が作成したデータがご自身の画面に反映されました。

---

## 2. 計画内容の確認（レビュー）

データが正しく入力されているか、認定要件を満たしているかを確認します。

1. サイドバーにある**「📊 進捗詳細を確認 (Dashboard)」ボタン**を押します。
2. **「Progress Dashboard」**画面が表示されます。
   * **認定可能性スコア**: 100点満点中、何点かを確認します。
   * **必須項目の達成率**: 未入力の必須項目がないかチェックします。
3. **不足項目の確認**:
   * 「申請に向けて、以下の必須項目が不足しています」という赤字のアラートが出ている場合は、その項目をメモします。
   * 具体的な中身は、画面下部のタブ（1️⃣ 基本情報 〜 6️⃣ 資金・推進体制）をクリックして閲覧できます。

---

## 3. 指導・修正のサポート

### 不足項目の洗い出し
ダッシュボードで確認した「未入力項目」や「具体性が不足している点（例：協力者が空欄など）」について、経営者にヒアリングを行ってください。

### 修正入力の代行（または指示）
職員様ご自身がアプリ上で追記入力することも可能です。

1. ダッシュボードから左上の「⬅️ **商工会職員インタビュー**」または「経営者インタビュー」ボタンでチャット画面に戻ります。
2. **「商工会職員」モード**で、不足していた情報を入力します。
   * 例: 「避難場所は〇〇ふれあいセンターとして設定してください」
   * 例: 「協力者として〇〇建設を追加します」
3. 入力後、再度ダッシュボードで「解析する」ボタンを押し、データが反映されたことを確認します。
   * スコアが上がり、アラートが消えれば修正完了です。

---

## 4. 最終データの書き出しと申請

計画が完成したら、最終版のデータを保存します。

1. ダッシュボード画面にて、再度「解析する」を実行し、最新状態にします。
2. 「💾 **計画データを保存 (JSON)**」ボタンでデータを保存します（バックアップ用）。
3. 必要に応じて「📄 **下書きシート出力 (Excel)**」を行い、Excel形式の申請書下書きを作成します。
4. このExcelファイルの内容をもとに、国（中小機構）の電子申請システムへ入力を行ってください。

---

**ヒント: 複数企業の支援について**
このアプリは1回のログインにつき1社分のデータを扱います。
別の企業の支援を行う場合は、ブラウザをリロード（再読み込み）して初期状態に戻してから、新しいJSONファイルを読み込ませてください。
