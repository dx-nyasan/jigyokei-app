# 事業継続力強化計画（BCP）策定アプリ 操作マニュアル (従業員編)

本マニュアルは、現場で働く従業員様が本アプリを使用して、事業継続力強化計画（BCP）の策定に協力するための手順書です。

---

## 🛑 はじめに：あなたの役割

### なぜ従業員の協力が必要なのですか？
経営者だけでは見えない「現場のリアルなリスク」や「具体的な手順」を計画に反映させるためです。
災害時に本当に役立つ計画を作るためには、実際に現場で働く皆さんの声が不可欠です。

### あなたのゴール
1. AIからの質問に答え、現場の状況（避難経路、緊急連絡網、設備の操作手順など）を伝える。
2. 業務マニュアルや現場の写真があれば、アプリに読み込ませる。
3. 担当部分の入力を完了し、経営者に報告する。

---

## 1. アプリへのログイン方法

1. ブラウザ（Google ChromeやEdgeなど）で、案内されたURLにアクセスします。
2. **パスワード入力画面**が表示されます。
   - 経営者から共有された「パスワード」を入力し、Enterキーを押してください。
3. アプリのトップ画面が表示されます。
4. 画面左上（またはメニュー）で**「従業員インタビュー」**が選択されていることを確認してください。
   - ※もし「経営者インタビュー」になっている場合は、画面左側の「サイドバー」から切り替えるか、または経営者に確認してください。

---

## 2. チャット（入力・会話）の方法

AIコンシェルジュが、現場の防災対策について質問します。普段の業務やお考えをそのまま入力してください。

### 入力のコツ
*   **正直に答える**: 「まだ決まっていません」「わかりません」という回答も重要です（何が足りないかが明確になるため）。
*   **具体的に**: 「〇〇を確認する」より「壁の掲示板にある緊急連絡網を見る」のように、具体的な行動を書くとより良い計画になります。

### 便利な機能
*   **クイック返信**: AIが提示する選択肢ボタンを押すだけで回答できます。
*   **回答例ボタン**: AIが良い回答例（緑色の枠）を出してくれることがあります。「これだ！」と思ったら、**「📋 回答例の通り回答する」ボタン**を押してください。

---

## 3. 資料のアップロード方法

手元に業務マニュアルや連絡網の表、現場の写真などがある場合は、アップロードしてください。AIがそれを読んで学習します。

1. 画面上部の「📂 資料の追加アップロード」を開きます。
2. 「👷 **現場・業務資料をアップロード**」という欄に、ファイルをドラッグ＆ドロップします。
   - 対応ファイル: PDF、画像（スマホで撮った写真もOK）
3. アップロードすると、AIが内容を分析し、「避難場所は〇〇小学校ですね」のように理解してくれます。

---

## 4. 進捗の確認と終了

### 進捗の確認
画面左側のサイドバーに出ている「現在の進捗バー」で、全体の進み具合を確認できます。

### 入力が終わったら
AIから「現場に関する質問は以上です」と言われたり、これ以上伝えることがなくなったりしたら、作業は完了です。

1. ログアウトや特別な保存操作は不要です（自動保存されます）。
2. **経営者（または管理者）に、「入力が終わりました」と報告**してください。
   - その後、経営者が内容を確認し、正式な計画書としてまとめます。
